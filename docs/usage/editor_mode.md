# Editor Mode

Editor Mode is an official version of Minecraft Bedrock on Windows designed for map making. Likewise there's a version of WorldEdit available with Editor Mode integration, taking advantage of its user interface for more seamless map making. This version can be downloaded in the same place as the regular version.

[TOC]

## Prerequisite

To use the editor mode version of WorldEdit, you need to have Editor Mode installed and know how to use it. Microsoft has official documentation for Editor mode that can be found [here](https://learn.microsoft.com/en-us/minecraft/creator/documents/bedrockeditor/editoroverview?view=minecraft-bedrock-stable).

## Features

WorldEdit with Editor Mode has the following features compared to the normal version.

- Undo and Redo using Shortcut keys (++ctrl+z++ and ++ctrl+y++)
- Editor brush controls (creating strokes by holding your mouse over terrain)
- Gradient management via UI
- Region operation management via UI
